OpenPV (The Open PV Project)

The real-time status of the solar photovoltaic market in the U.S.

The Open PV Project provides search, data and visualization gallery information.

Installs Index API (GET /api/solar/open_pv/installs/index)

Returns a list of all installs matching optional criteria passed or all installs if no optional criteria is passed. The response can be paged or exported.

Installs Rankings API (GET /api/solar/open_pv/installs/rankings)

Returns a list of install costs and counts by state.

Installs Summaries API (GET /api/solar/open_pv/installs/summaries)

Returns a list of install summary data filtered by optional parameters.
